# SCAMPER Agent

[English](README.md) | [正體中文 (zh-TW)](README.zh-TW.md) | [简体中文 (zh-CN)](README.zh-CN.md)

SCAMPER Agent 是一个 AI Agent 技能，基于 SCAMPER 技巧引导用户进行结构化的头脑风暴与评估流程。当面临复杂问题、探索创新替代方案，或是寻求对产品、服务或流程的改善时，很容易在缺乏结构的头脑风暴中迷失方向。SCAMPER Agent 通过引导您进行严谨、系统化的冲刺，确保每个想法都能被产生、追踪并系统性地评估。具体来说，它利用 SCAMPER 法来进行创意发想，并利用 ICE 评分法来进行创意筛选与优先级排序。

## 方法论 (Methodology)
SCAMPER Agent 使用分为 4 个阶段的工作流程，将概略的目标转化为可执行的策略：
1. **厘清目标与限制 (Clarify Goals & Constraints)：** 从用户的输入中推断主要目标、预算、时间表及限制，然后提出配置建议以供确认。
2. **由 SCAMPER 进行头脑风暴 (Brainstorming by SCAMPER)：** 有条理地产生被索引的想法，且严格对齐 SCAMPER 框架。
3. **通过 ICE 进行想法筛选 (Idea Screening by ICE)：** 通过限制条件与目标契合度检查过滤想法（移除、保留或标记为有条件保留），然后通过计算 ICE 分数严格排序合格的想法。
4. **战略性建议报告 (Strategic Recommendation Report)：** 将结果汇总为一份战略报告，包含执行路线图、资源分配计划、建议的最小可行产品 (MVP) 以及被排除想法的附录。

### SCAMPER
一种用来激发新想法的创意思考技巧，通过 7 个维度提出针对性问题：

| 字母 | 维度 | 描述 |
| :---: | :--- | :--- |
| **S** | **替代 (Substitute)** | 哪些组件、材料或流程可以被替换？ |
| **C** | **结合 (Combine)** | 哪些不相关的技术或服务能够被合并？ |
| **A** | **调适 (Adapt)** | 可以借用其他领域的哪些原理？ |
| **M** | **修改 (Modify)** | 特征、规模或频率如何改变？（包含放大/缩小） |
| **P** | **其他用途 (Put to another use)**| 核心机制还能应用在哪些其他地方？ |
| **E** | **消除 (Eliminate)** | 移除什么可以创造更精简、更快速的版本？ |
| **R** | **反转 (Reverse)** | 通过反转流程或结构，会出现什么新方法？ |

### ICE 评分 (ICE Scoring)
一个优先级排序框架，基于三个标准评估保留的想法，以确保一致的排名：

| 指标 | 评估内容 |
| :---: | :--- |
| **影响力 (Impact)** | 若成功所能带来的潜在价值；与主要目标的契合度。 |
| **信心 (Confidence)** | 基于现有知识与证据能成功的概率。 |
| **容易度 (Ease)** | 考量现有资源与时间表，执行的简易程度。 |

每个指标的分数范围从 1 到 10 分，使用以下校准标准：

| 分数 | 影响力 (Impact) | 信心 (Confidence) | 容易度 (Ease) |
| :---: | :--- | :--- | :--- |
| 1–2 | 微不足道的改善 | 纯属推测，无佐证资料 | 需要重大的新能力或基础设施 |
| 3–4 | 轻微的使用体验提升 | 理论可行但未经测试 | 重大精力投入；取决于外部因素 |
| 5–6 | 具体、可衡量的改善 | 有间接证据或类比，具有合理性 | 目前团队能力可及的适度努力 |
| 7–8 | 强大的战略优势 | 由数据、案例研究或过往成功经验支持 | 使用现有工具与技能即可直接执行 |
| 9–10 | 改变游戏规则或定义市场 | 高确定性且经过验证的方法 | 使用现有资源，数日内即可发布 |

## 示例问题 (Sample Questions)

### 1. 产品开发
当您想要为实体或数字产品构思新功能，或是进行彻底的重新构想时。

> *"我们想为户外运动爱好者打造一款次世代的智能水壶。请使用 SCAMPER 法来激发关于新功能、材质和外型的灵感，然后使用 ICE 评分法评估出最好的点子。"*

### 2. 服务创新
当您希望在服务业中重塑客户体验或提供新的价值主张时。

> *"我们的精品酒店想要彻底重塑入住 (check-in) 和退房 (check-out) 的体验。请带领我们进行一次 SCAMPER 头脑风暴，产出创新的服务点子，并将重点放在消除摩擦以及调适最新的旅宿趋势。"*

### 3. 流程改善
当内部工作流程缺乏效率、成本过高或速度太慢，而您需要有创意的方法来进行优化时。

> *"我们目前的员工入职流程 (onboarding process) 高度依赖人工操作，占用资深员工太多时间，但新进人员在公司文化上仍感到脱节。请使用 SCAMPER 法来发想彻底改造组织入职流程的创意方法，并以 ICE 评分严格筛选出最可行的方案。"*

## 使用方式 (How to Use)
您可以将此 agent 设置整合到 Gemini Gems 中（使用 [`gemini-gem.md`](gemini-gem.md)），或是若您想将其作为可用的技能 (skill) 使用，请参考 [`scamper-agent/`](scamper-agent/) 目录。

您可以通过在提示词中包含关键字或短语来触发 agent，例如：
- *"SCAMPER"*, *"ICE 评分"*, *"创新"*, *"发想点子"*, *"我该选择哪个想法"*,
- 或任何要求系统性探索与限缩选项的请求。
