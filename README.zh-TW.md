# SCAMPER Agent

[English](README.md) | [正體中文 (zh-TW)](README.zh-TW.md) | [简体中文 (zh-CN)](README.zh-CN.md)

SCAMPER Agent 是一個 AI Agent 技能，基於 SCAMPER 技巧引導使用者進行結構化的腦力激盪與評估流程。當面臨複雜問題、探索創新替代方案，或是尋求對產品、服務或流程的改善時，很容易在缺乏結構的腦力激盪中迷失方向。SCAMPER Agent 透過引導您進行嚴謹、系統化的衝刺，確保每個想法都能被產生、追蹤並系統性地評估。具體來說，它利用 SCAMPER 法來進行創意發想，並利用 ICE 評分法來進行創意篩選與優先級排序。

## 方法論 (Methodology)
該 agent 的工作流程基於兩個經過驗證的框架：

### SCAMPER
一種用來激發新想法的創意思考技巧，透過 7 個維度提出針對性問題：

| 字母 | 維度 | 描述 |
| :---: | :--- | :--- |
| **S** | **替代 (Substitute)** | 哪些元件、材料或流程可以被替換？ |
| **C** | **結合 (Combine)** | 哪些不相關的技術或服務能夠被合併？ |
| **A** | **調適 (Adapt)** | 可以借用其他領域的哪些原理？ |
| **M** | **修改 (Modify)** | 特徵、規模或頻率如何改變？（包含放大/縮小） |
| **P** | **其他用途 (Put to another use)**| 核心機制還能應用在哪些其他地方？ |
| **E** | **消除 (Eliminate)** | 移除什麼可以創造更精簡、更快速的版本？ |
| **R** | **反轉 (Reverse)** | 透過反轉流程或結構，會出現什麼新方法？ |

### ICE 評分 (ICE Scoring)
一個優先級排序框架，基於三個標準評估保留的想法，以確保一致的排名：

| 指標 | 評估內容 |
| :---: | :--- |
| **影響力 (Impact)** | 若成功所能帶來的潛在價值；與主要目標的契合度。 |
| **信心 (Confidence)** | 基於現有知識與證據能成功的機率。 |
| **容易度 (Ease)** | 考量現有資源與時間表，執行的簡易程度。 |

每個指標的分數範圍從 1 到 10 分，使用以下校準標準：

| 分數 | 影響力 (Impact) | 信心 (Confidence) | 容易度 (Ease) |
| :---: | :--- | :--- | :--- |
| 1–2 | 微不足道的改善 | 純屬推測，無佐證資料 | 需要重大的新能力或基礎設施 |
| 3–4 | 輕微的使用體驗提升 | 理論可行但未經測試 | 重大精力投入；取決於外部因素 |
| 5–6 | 具體、可衡量的改善 | 有間接證據或類比，具有合理性 | 目前團隊能力可及的適度努力 |
| 7–8 | 強大的策略優勢 | 由數據、案例研究或過往成功經驗支持 | 使用現有工具與技能即可直接執行 |
| 9–10 | 改變遊戲規則或定義市場 | 高確定性且經過驗證的方法 | 使用現有資源，數日內即可發布 |

## 運作方式 (How It Works)
SCAMPER Agent 使用分為 4 個階段的工作流程，將概略的目標轉化為可執行的策略：
1. **釐清目標與限制 (Clarify Goals & Constraints)：** 從使用者的輸入中推斷主要目標、預算、時間表及限制，然後提出配置建議以供確認。
2. **SCAMPER 腦力激盪 (SCAMPER Brainstorming)：** 有條理地產生被索引的想法，且嚴格對應 SCAMPER 框架。
3. **想法篩選 (Idea Screening)：** 透過限制條件與目標契合度檢查過濾想法（移除、保留或標記為有條件保留），然後透過計算 ICE 分數嚴格排序合格的想法。
4. **策略性建議報告 (Strategic Recommendation Report)：** 將結果統整為一份戰略報告，包含執行藍圖、資源分配計畫、建議的最小可行產品 (MVP) 以及被排除想法的附錄。

## 範例問題 (Sample Questions)

### 1. 產品開發
當您想要為實體或數位產品發想新功能，或是進行徹底的重新構想時。

> *"我們想為戶外運動愛好者打造一款次世代的智慧水壺。請使用 SCAMPER 法來激發關於新功能、材質和外型的靈感，然後使用 ICE 評分法評估出最好的點子。"*

### 2. 服務創新
當您希望在服務業中重塑客戶體驗或提供新的價值主張時。

> *"我們的精品旅館想要徹底重塑入住 (check-in) 和退房 (check-out) 的體驗。請帶領我們進行一次 SCAMPER 腦力激盪，產出創新的服務點子，並將重點放在消除摩擦以及調適最新的旅宿趨勢。"*

### 3. 流程改善
當內部工作流程缺乏效率、成本過高或速度太慢，而您需要有創意的方法來進行最佳化時。

> *"我們目前的員工入職流程 (onboarding process) 高度仰賴人工作業，佔用資深員工太多時間，但新進人員在公司文化上仍感到脫節。請使用 SCAMPER 法來發想徹底改造組織入職流程的創意方法，並以 ICE 評分嚴格篩選出最可行的方案。"*

### 使用方式 (How to Use)
您可以將此 agent 設定整合到 Gemini Gems 中（使用 [`gemini-gem.md`](gemini-gem.md)），或是若您想將其作為可用的技能 (skill) 使用，請參考 [`scamper-agent/`](scamper-agent/) 目錄。

您可以透過在提示詞中包含關鍵字或片語來觸發 agent，例如：
- *"SCAMPER"*, *"ICE 評分"*, *"創新"*, *"發想點子"*, *"我該選擇哪個想法"*,
- 或任何要求系統性探索與限縮選項的請求。
